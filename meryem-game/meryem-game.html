--- # **VISUAL STUDIO CODE İLƏ NECƏ DÜZƏLT MƏ (ADIM-ADIM)** ## **1️⃣ Faylı yaratmaq** 1. Kompütərdə **Visual Studio Code** aç. 2. Sol tərəfdə **Explorer** → **New File**. 3. Faylın adını yaz: **meryem.html** 4. Canvasdakı kodu olduğu kimi kopyalayıb bu fayla yapışdır. 5. **Ctrl + S** ilə yadda saxla. --- ## **2️⃣ Brauzerdə açmaq** ### Variant A — Birbaşa açma (ən asan yol) 1. Kompüterdə **meryem.html** faylının üzərinə iki dəfə klik et. 2. Brauzerdə açılacaq. 3. Telefonuna göndərmək üçün faylı WhatsApp/Telegram ilə göndərə bilərsən. ### Variant B — Live Server (VS Code içindən) 1. VS Code-da **Extensions** bölməsinə gir. 2. "**Live Server**" yaz və quraşdır. 3. meryem.html aç → sağ klik → **Open With Live Server** 4. Sayt avtomatik brauzerdə açılacaq. --- # **3️⃣ Telefonla paylaşmaq** ### **EN ASAN YOL** Faylı sevgilinə göndər: - WhatsApp - Telegram - e-mail - Google Drive linki - AirDrop (iPhone) O, sadəcə faylı açıb oynayacaq. --- # **4️⃣ Linklə paylaşmaq (istəsən online sayt kimi)** ### **GitHub Pages (tam pulsuz, linklə açılır)** 1. github.com-da hesab aç. 2. New Repository → ad: **meryem-oyunu**. 3. İçinə **meryem.html** faylını yüklə. 4. Settings → Pages → Source = **main branch** → Save. 5. 1-2 dəqiqəyə link hazır olur: `https://istifadəçi-adın.github.io/meryem-oyunu/meryem.html` Bu linki Meryemə göndərirsən və oyun internetdə açılır ❤️
